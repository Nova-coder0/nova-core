## Submission Requirements To Resolve

BscScan profile updates normally require:

- verified contract ownership,
- published source code verification where possible,
- an accessible website with clear project information,
- a public logo link,
- working social or documentation links,
- neutral project wording without exaggerated price or performance claims.
